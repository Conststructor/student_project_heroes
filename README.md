### SimulateBattleImpl
Метод осуществляет симуляцию боя между армией игрока и армией компьютера.

**Внешний цикл do-while**
Условие завершения: пока одна из армий не станет пустой
***Худший случай: O(n * m), где:***
n - максимальное количество юнитов в армии (playerArmySize или computerArmySize)
m - количество полных циклов боя до полного уничтожения одной из армий

**Операции внутри цикла:**
Проверка размера армии: O(1)
Атака и вызов метода attack(): O(1)
removeIf() для удаления юнитов: O(k), где k - количество юнитов в армии
